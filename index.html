KEYBOARD SHORTCUTS ( Tested On WINDOWS ^_^ )

  1. ctrl + window key + z to undo
  2. Left and Right arrow keys for previous and next frame
  3. ctrl + alt + p to enter presentation mode
  4. esc to escape any tool or presentation mode
  5. t for text tool
  6. c for camera
  7. ctrl + win key + backspace key(<-) to delete any selected objects
  8. ctrl + win + click and drag to pan when camera is active
  9. ctrl + alt + (I L K J) to scale selected object
  10. ctrl + alt + (U O) to rotate selected object

THINGS TO DO WITH TEXT BOXES

  1. "x=5" to assign a value to a variable
  2. "e:x=5" to repeatedly update x every frame
  3. "f(x)=sin(x)" to define a function
  4. "e:expression" to run expression each frame
  5. "slide:var_name" then present, and you can slide the box to change the variable value
  6. "e:graph(function_name)" to graph function_name(x)

ANIMATION BASICS

  1. Create a text or circle or drawing
  2. Change frames with the left and right arrow keys
  3. Change the text or object's properties on the new frame
  4. Enter presentation mode by hitting "p"
  5. Left and Right arrow keys will show animated transition
  6. Escape presentation mode with ESC key

CONTROL CAMERA

  1. Click camera or hit "c".
  2. Click and drag to pan.
  3. ctrl + win + click and drag to rotate.
  4. The camera will transition smoothly frame to frame.

FILES

  1. Save local will overwrite a local file in browser's cache.
  2. Load local clears your work and loads the cache file.
  3. To save enter a filename above and hit "Save File".
  4. To load hit load file and select your saved file.
  5. As you edit if you make a mistake you can undo by hitting ctrl + z.

MORE FUNCTIONS

  1. "L=range(a,b,c)" creates array of values from a to b increment c.
  2. "e:view(L, [x,y,z])" print matrix elements at position [x,y,z].
  3. "e:surface(func)" to graph y=func(x,z) as a grid of points.
  4. "e:scatter(A)" A is n by 3 matrix of points to be scatter plotted.
  5. "A=randn([a,b,c,...])" to create matrix of random normal values.
  6. "A[1,2]" to index into matrix A with 1 based indexing, not zero! :-(

SPECIAL VALUES

  1. _x mouse x position, _y mouse y position.
  2. _frame current frame (counts up indefinitely).
  3. _t frame transition from 0 to 1, _tt eased transition.
